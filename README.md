# HackTheBox Writeups 🎯

Welcome to my personal collection of writeups, notes, and documentation for **Hack The Box** machines and challenges. This repository serves as a diary of my cybersecurity journey, tracking my progress as I develop my skills in penetration testing, vulnerability assessment, and security research.

Because let's honest: remembering exactly how you rooted an obscure Windows Active Directory box six months ago is harder than the exploit itself.

## 📂 Repository Structure

The writeups are organized logically to make navigation seamless:

*   **/Machines** - Step-by-step guides for full OS boxes, categorized by difficulty (Easy, Medium, Hard, Insane) and OS (Linux, Windows).
*   **/Challenges** - Solutions for standalone challenges (Crypto, Reverse Engineering, Pwn, Web, Forensics).
*   **/Scripts-and-Payloads** - Custom scripts or modified exploits used during the boxes.

## 🛠️ Methodology Highlighted

Every writeup focuses heavily on *methodology* rather than just pasting the flags. You'll find detailed breakdowns of:
1. **Information Gathering & Enumeration** (`nmap`, `gobuster`, `nikto`, etc.)
2. **Vulnerability Analysis** (Identifying misconfigurations, outdated software, or weak credentials)
3. **Exploitation / Foothold** (Crafting payloads, weaponizing exploits)
4. **Privilege Escalation** (Local enumeration, pivoting, and gaining `root`/`SYSTEM`)
